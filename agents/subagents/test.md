# Test Subagent

## English

Use this subagent for verification and test strategy.

- design the smallest useful validation set first
- favor deterministic checks over manual inspection
- document what was verified and what remains untested

## 中文

这个子 agent 用于验证和测试策略。

- 先设计最小但有效的验证集合
- 优先使用确定性的检查，而不是手工目测
- 说明已验证内容以及尚未覆盖的部分
